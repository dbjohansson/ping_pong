Overview

The Ping Pong game is developed in Dart to start learning Dart (after reading the Getting Started section and developing the first example). The spiral approach to learning is used to advance step-by-step. The game is based on the Bounce game developed by Bill Mill in JavaScript and Chris Buckett in Dart.

The code is kept at GitHub. If you are new to Git, you should really start learning it.

Spiral 1

Display a small, black circle and a small, white rectangle with a black border.

Spiral 2

Display two circles and two rectangles with different colors.

Spiral 3

Change a position of a small, black circle (a ball) in the same direction, with each new call of the move method.

Spiral 4

Reorganize the code a bit.

Spiral 5

Bounce the ball within the board.

Spiral 6

Display a racket (a small, black rectangle).

Spiral 7

Move the racket with the left and right keys of the keyboard.

Spiral 8

Move the racket left or right with the mouse.

Spiral 9

The board with a border and two rackets. The rackets cannot be moved outside the border. The game can be played.

Spiral 10

The game has a title and a button to restart the game.

Spiral 11

The page uses CSS for a better display. There are some relevant web links. A section is added for each new spiral to indicate what is new.
